# golden6-bot

## 簡介

基於[LINE MESSAGING API](https://developers.line.biz/zh-hant/docs/messaging-api/) + [Firebase](https://firebase.google.com/)的接龍機器人

只需要傳

> Re (或re/回報)
> 
> (必填，在哪裡)
> 
> (必填，跟誰)
> 
> (選填，做甚麼)

即可自動彙整該班的消息，拼成接龍


## 直接使用

1. 添加golden6為好友 https://page.line.me/657nmkau 
2. 把它加到群組中
